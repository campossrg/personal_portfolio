Integración tarjeta T-Mobilitat

T-Mobilitat es el proyecto de integración de la tarjeta contacless para unificar todos los medios de transporte de Cataluña. De esta manera se promueve el uso de un único medio de pago tipo monedero recargable en cualquier punto de venta. El sistema se gestiona a través de servidores de aplicaciones interconectando cada operador de transporte. En cada uno se integran modulos que administran las diversas funcionalidades: Topología de red de transporte, equipos de campo, titulos de transporte, tarifas, envío de configuraciones y listas de acciones, etc.
